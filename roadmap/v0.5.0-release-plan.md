# Alopex v0.5.0 / alopex-py v0.5.0 リリース計画

**最終更新**: 2026-01-13

> **Note**: alopex-py は Rust ワークスペースとは独立したバージョン体系で管理されています。
> このドキュメントでは両プロジェクトの v0.5.0 リリースに向けた作業計画をまとめています。

---

## 概要

### Alopex DB v0.5.0

**テーマ**: Durability + JOIN Support + DataFrame P1

v0.4.0 で実装された Server モードと Async/Stream 基盤の上に、データ永続性の強化と SQL JOIN 機能を追加します。
また、alopex-dataframe v0.2.0 で DataFrame API の結合・ソート機能（P1）を実装します。

### alopex-py v0.5.0

**テーマ**: DataFrame API Python バインディング（P0 + P1）

alopex-dataframe v0.2.0 の機能を Python から利用可能にする DataFrame API バインディングを実装します。

## リリース順序

```
1. alopex-core v0.5.0 (Durability 強化)
   ↓
2. alopex-sql v0.5.0 (GROUP BY + JOIN)
   ↓
3. alopex-dataframe v0.2.0 (P1: 結合・ソート)
   ↓
4. alopex-embedded v0.5.0 (統合)
   ↓
5. alopex-server v0.5.0 (API 対応)
   ↓
6. alopex-py v0.5.0 (DataFrame Python バインディング)
```
